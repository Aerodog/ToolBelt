ToolBelt by Peter200lx

Play Minecraft with a wide assortment of tools at your side.

ToolBelt provides a wide range of tools for the discerning minecraft player. Ranging from easily duplicating a selected block, to leaping through the air with style.

Binaries available at: http://dev.bukkit.org/server-mods/toolbelt/

Dependencies

http://dl.bukkit.org/

Install

If you haven't already, you will need to install CraftBukkit. You can get that software and instructions on its use at: http://wiki.bukkit.org/Setting_up_a_server

The process for setting up this mod is to drop it in your CraftBukkit plugins/ folder. Run (or /reload) bukkit, and it will create a ToolBelt/ folder with the config.yml file inside. Modify the file as wished (it is documented inside the file) and then run /tb reload to tell ToolBelt to reconfigure itself from modified config file.

Most up-to-date instructions available at: http://dev.bukkit.org/server-mods/toolbelt/
